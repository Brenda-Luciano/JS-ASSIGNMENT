6. Explain the difference between == and ===.

The operator == means equal value and the === operator means equal value and equal type.

7. When would you receive a NaN result?

You receive a NaN result when the value can't be computed or is an invalid number.

8. How do you increment and decrement a number?

To increment a number we use the operator ++ and to decrement a number we use the operator --.

9. Explain the difference between prefixing and post-fixing increment/decrement operators.

Increment: with Prefix, the increment operator adds and returns the value after the incrementing. With Postfix, the increment operator adds and returns the value before the increment.

Decrement: with Prefix, the increment operator subtracts and returns the value after the decrementing. With Postfix, the increment operator subtracts and returns the value before the decrementing.

10. What is operator precedence and how is it handled in JS?

Operator precedence determines how operators are ordered; operators with higher precedence are evaluated first and become the operands of operators with lower precedence.

11. How do you log information to the console?
